# COVID-19 Deaths and Vaccinations Analysis

## Overview

This project explores global COVID-19 trends using publicly available data on cases, deaths, and vaccinations. The goal of the analysis was to better understand how the pandemic affected different countries over time and compare vaccination progress across regions.

The data was cleaned and analyzed using SQL, then visualized in Tableau through an interactive dashboard.

---

## Tools Used

- SQL
- Tableau
- Excel

---

## Project Goals

This analysis focused on questions such as:

- Which countries had the highest infection and death rates?
- How did vaccination rollout differ across countries?
- What trends appeared throughout different stages of the pandemic?
- How did total cases compare relative to population size?

---

## Data Cleaning and Preparation

The datasets were cleaned and prepared using SQL before being imported into Tableau for visualization.

Some of the steps included:
- Removing unnecessary columns
- Standardizing data formats
- Handling missing values
- Creating calculated metrics
- Joining deaths and vaccination datasets
- Calculating rolling vaccination totals

---

## Tableau Dashboard

An interactive Tableau dashboard was created to visualize key COVID-19 trends and vaccination progress around the world.

Dashboard features include:
- Global case and death trends
- Vaccination progress over time
- Country-level comparisons
- Interactive filtering by region and date

## Key Insights

- COVID-19 impacted countries differently in terms of infection rates and mortality.
- Vaccination rollout speeds varied significantly across regions.
- Some countries experienced high case counts with relatively lower death percentages.
- Time-based visualizations helped highlight major pandemic waves and vaccination milestones.
